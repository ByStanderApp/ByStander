ชื่อเคส: ไข้สูงชัก
Keywords: เด็กชัก, ตาเหลือก, ตัวร้อนแล้วชัก, เกร็งกระตุก, กัดลิ้น
Severity: critical
Facility: Hospital

แนวทางปฐมพยาบาล:
1. จับให้นอนตะแคง ไม่หนุนหมอน หันศีรษะไปด้านใดด้านหนึ่งเพื่อป้องกันการสำลัก
2. คลายเสื้อผ้าให้หลวม
3. ห้ามใช้ช้อนหรือวัตถุอื่นๆ หรือนิ้วมืองัดปาก และห้ามป้อนยาหรือน้ำทางปากในขณะที่เด็กไม่รู้สึกตัว
4. เช็ดตัวด้วยน้ำจากก๊อกประปาหรือน้ำอุ่น ไม่ควรเช็ดตัวด้วยน้ำเย็นหรือแอลกอฮอล์
5. นำเด็กส่งโรงพยาบาลหรือรีบไปพบแพทย์ เพื่อให้แน่ใจว่าสาเหตุของการชักเกิดจากไข้สูง หรือจากสาเหตุอื่นๆ เช่น การติดเชื้อในสมอง ซึ่งต้องรักษาที่ต้นเหตุด้วย
